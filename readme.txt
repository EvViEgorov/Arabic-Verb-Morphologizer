# Arabic Verb Morphologizer
Морфологизатор арабского глагола в виде HTML-страницы c использованием PyScript для анализа грамматических форм сильных и слабых глаголов. Функционал программы позволяет:
1. Генерировать парадигму (таблицу всех глагольных форм) для любого трёхбуквенного арабского корня;
2. Анализировать введённую словоформу и определять возможные характеристики.

# Структура репозитория:
Имеется две основные ветки:
1. main - главная ветка, содержит файлы, необходимые для работы сайта
2. develop - ветка для разработки, содержит разделённые на файлы функции, а также функции в разработке
